# PanoramaBroadcast
PanoramaBroadcast是基于 Oculus DK1 的全景序列帧的播放系统，使用C/C++结合OpenGL和GLSL编写

使用到的库：FreeImage，glfw-3.1.1，glm-0.9.4.0，OculusSDK0.3.2

编译环境：vs2010
